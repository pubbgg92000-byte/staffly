# Staffly

Staffly is the working repository for PeopleFlow, a multi-tenant HRMS for SMBs.

Sprint 0 is intentionally infrastructure-only:

- Monorepo scaffolding
- Next.js admin shell
- Next.js employee shell
- NestJS API foundation
- PostgreSQL and Prisma foundation
- Auth and RBAC skeletons
- Shared UI, types, config, and i18n packages

Feature modules such as employee CRUD, attendance, leave, documents, announcements, and holidays start after the foundation is stable.
